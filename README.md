# Memory Game

Memory Game sviluppato usando la libreria [Raylib](https://www.raylib.com/).

---

## Requisiti

- CMake version >= 3.30  
- Visual Studio 2022 (version 17) su Windows  
- GCC/Clang su Linux  
- Raylib (le librerie sono incluse nel progetto)

---
## Installation

### Linux

```bash
chmod +x build.sh
```
### Windows
```bash
./build.sh -a x86 -t Release
```
